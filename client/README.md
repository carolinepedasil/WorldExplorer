# WorldExplorer Client

Frontend application for WorldExplorer built with Next.js, React, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Data Fetching**: React Query (@tanstack/react-query)
- **HTTP Client**: Axios
- **State Management**: Zustand
- **Routing**: Next.js App Router (built-in)

## Features

- Modern Next.js App Router architecture
- Server-side rendering and client components
- GitHub OAuth authentication
- Email/password authentication
- JWT token management
- Protected routes
- Responsive design with Tailwind CSS
- Dark mode support
- Type-safe API calls

## Getting Started

First, install dependencies:

```bash
npm install
```

Configure environment variables by creating a `.env.local` file (already provided).

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
